My Technical Journey

What I Learned

Creating this website has been a transformative experience. I needed to learn a variety of web development technologies, including:

  • HTML: To structure the website and create a solid foundation.
  • CSS: To design an attractive and user-friendly interface.
  • JavaScript: To add interactivity, such as the Zodiac Calculator and Virtual Kitchen game.
  • Responsive Design: To ensure the site looks good on both desktop and mobile devices.

Challenges I Faced

There were several challenges along the way:

  • Embedding Videos: Initially, I struggled to embed videos on the festival pages. After researching online, I learned about the <video> tag and successfully added small, responsive videos.
  • Interactive Features: Implementing the Zodiac Calculator was a significant challenge. Understanding how to use JavaScript functions for user input required me to read “Eloquent JavaScript” by Marijn Haverbeke, which provided clear examples.
  • Navigation Bar: Ensuring the navigation bar worked seamlessly across all pages was tricky. I found a solution by creating a shared CSS style and copying the structure to each HTML file.

Solutions I Found

Each challenge was an opportunity to learn and grow. Here are some specific solutions I used:

  • Video Tutorials: I followed several YouTube tutorials on HTML and CSS. One particularly helpful video was "Responsive Web Design for Beginners."
  • Debugging Tools: Using the browser's developer tools allowed me to identify and fix layout issues quickly.
  • Online Resources: Websites like MDN Web Docs and W3Schools were invaluable for learning new HTML and CSS features.

Key Takeaways

This project taught me the importance of persistence and resourcefulness. Whether it was understanding flexbox layouts or debugging JavaScript code, I learned to break problems into smaller parts and tackle them one at a time.
